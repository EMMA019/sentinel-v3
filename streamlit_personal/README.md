# SENTINEL PRO — 個人用フル機能版

## 公開サイトとの違い
- 株価・ローソク足チャート（mplfinance）を**価格直接表示**
- エントリー/ストップ/ターゲット価格を**ドル表示**
- アナリスト目標株価（$表示）
- 全銘柄スキャン結果のCSVダウンロード
- ウォッチリスト管理

## 起動方法
```bash
cd streamlit_personal
pip install -r requirements.txt
FMP_API_KEY=あなたのAPIキー streamlit run app.py
```

このファイルは**個人利用専用**です。公開・配布はFMPの規約違反になります。
